# LiteratureHub

A simple website showcasing literature, authors, and stories. Built with HTML and CSS.

## Pages

- **Home:** `index.html` – Main page
- **Authors:** `authors.html` – Information about authors
- **Stories:** `stories.html` – Collection of stories
- **Contact:** `contact.html` – Contact page
- **About:** `about.html` – About the website
- **Tips:** `tips.html` – Useful tips
- **CSS:** `literaturehub.css` – Styles for the website

## Live Demo

Check it out online: [https://shaira-del.github.io/literaturehub/](https://shaira-del.github.io/literaturehub/)

## How to Use

1. Open any of the `.html` files in a browser.
2. Navigate through the menu to explore different pages.
